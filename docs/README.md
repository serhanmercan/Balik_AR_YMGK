3D Balık AR Uygulaması

Bu proje, Fırat Üniversitesi Yazılım Mühendisliği bölümü Yazılım Mühendisliğinde Güncel Konular (YMGK) dersi kapsamında geliştirdiğim bir Artırılmış Gerçeklik (AR) uygulamasıdır.

Projenin amacı, kullanıcıların mobil cihaz kameraları aracılığıyla fiziksel çevrelerindeki düz zeminleri algılayıp, bu alanlar üzerinde 3 boyutlu balık modellerini görüntülemesini sağlamaktır. Proje, gerçek ortamda sistem prototipi olan THS 7 seviyesini hedeflemektedir.

Projede oyun motoru olarak Unity, programlama dili olarak C# ve AR altyapısı olarak AR Foundation kullanılmaktadır.

docs klasörünün içinde proje sürecindeki tasarım ve analiz belgeleri bulunmaktadır:
